CLE 1: BirdBot In dit project moest ik een hulpmiddel maken voor een medestudent. Dit hulpmiddel moest het dagelijks leven voor het persoon verbeteren. Ik heb voor dit project ook een website gemaakt. Hier kan je meer informatie vinden over het project en de teamleden.

CLE 2: Reserveringssysteem Albatros Barbershop Voor dit project heb ik een reserveringssysteem gemaakt voor het bedrijf Albatros Barbershop in 's-Gravendeel. In dit systeem kan je als klant een afspraak maken en als medewerker kan je inloggen om de gemaakte afspraken te bekijken.
Jaar1: OP2 HTML PHP
CLE 3: Eten op Rolletjes In dit project heb ik een webservice gemaakt voor een bepaalde doelgroep met een bijbehorende locatie. Mijn team en ik hadden de doelgroep: mensen met een motorische beperking en als locatie hadden wij het restaurant. De website heet Eten op Rolletjes. Er is veel informatie te vinden over hoe het vervoer en restaurants toegankelijk worden gemaakt voor mensen met een motorische beperking.
Jaar1: OP3 HTML Tailwind CSS
CLE 4: HofRunner In dit project heb ik met een groep gewerkt aan een game genaamd HofRunner. Dit is een Endless Runner waar je kan springen en verschillende power-ups kan oppakken. Ik heb gewerkt aan het designen van het spel en heb ik verschillende sprites gemaakt. Het design van de game heb ik gedaan in de stijl van 2D-pixelart.
Jaar1: OP4 Excalibur 2D Pixel Art

Jaar1: OP4 Excalibur JavaScript
Laravel CRUD Systeem In dit systeem kunnen gebruikers schilderijen toevoegen. Deze schilderijen kunnen gefilterd worden op artiest naam, naam schilderij en genre. In dit systeem zit een admin en gebruiker rol, waarbij de admin meer machten heeft dan de gebruiker. Ook zit er veel bevijliging in.
Jaar2: OP1 Laravel PHP Blade
TLE 1: AllergieScannerMEE In deze webapplicatie kunnen gebruikers etiketten van producten scannen. De gebruiker krijgt door het product te scannen informatie over het product (allergieën, calorieën etc.). Ik heb tijdens dit project gewerkt aan de UI en UX design, dus het uiterlijk van de webapplicatie. Onze opdrachtgever voor dit project was Mee Rotterdam Rijnmond.
Jaar2: OP1 - 2 Vue JavaScript Tailwind CSS
Inspiration AI Dit is een AI tool/chatbot die ideeën genereert (bijvoorbeeld voor kunstenaren) op basis van verschillende keywords/prompts die gebruiker mee kan geven. De AI tool genereert dan een idee of geeft verder inspiratie voor een kunstwerk (setting, style etc.) die de gebruiker kan gebruiken bij het maken van een kunstwerk.

CLE Projecten

Projecten

CLE 1 In dit project moest ik een hulpmiddel maken voor een medestudent. Dit hulpmiddel moest het dagelijks leven verbeteren voor de persoon. Ik heb voor dit project ook een website gemaakt. Hier kan je meer informatie vinden over het project en de teamleden.
Jaar1: OP1
CLE 2 Voor dit project heb ik een reserveringssysteem gemaakt voor het bedrijf Albatros Barbershop in 's-Gravendeel. In dit systeem kan je als klant een afspraak maken en als medewerker kan je inloggen om de gemaakte afspraken te bekijken.
Jaar1: OP2
CLE 3 In dit project moest ik een webservice maken voor een bepaalde doelgroep met een bijbehorende locatie. Ik had de doelgroep: mensen met een motorische beperking en als locatie: restaurant. Ik heb voor dit project een informative website gemaakt (Eten op Rolletjes). Er is veel informatie te vinden over hoe het vervoer toegankelijk wordt gemaakt voor mensen met een motorische beperking. Jaar1: OP3
CLE 4 In dit project heb ik met een groep gewerkt aan een game genaamd HofRunner. Dit is een Endless Runner waar je kan springen en verschillende power-ups kan oppakken. Ik heb gewerkt aan het designen van het spel en heb ik verschillende sprites gemaakt. Het design van de game heb ik in de stijl gedaan van 2D-pixelart. Jaar1: OP4
SpaceJourney Dit is een spel dat ik tijdens het vak "Programmeren4" gemaakt heb. SpaceJourney is een spaceshooter waarbij je zolang moet blijven overleven door de vijandelijke ruimteschepen te vernietigen. Ook kan je munten verzamelen om je score te verhogen. Jaar1: OP4
